# 浏览器工作台界面验收清单

适用范围：`src/mant/observability/dashboard.py` 内嵌的浏览器翻译工作台单页
（浅色简约版：浅灰白背景 `#f7f8fa` + 纯白面板 + 细边框 + 蓝色 `#2563eb`
强调）。本清单只覆盖界面与交互表现，功能与接口保持不变。

## 验收方法

1. 自检：在仓库根目录执行 `cmd /c start_mant.bat --check`，确认环境自检通过
   （自检不会打印任何密钥）。
2. 启动：双击 `start_mant.bat`，浏览器自动打开 `http://127.0.0.1:8765`。
3. 离线端到端：把 `data/raw/demo_work/src/0001.txt` 拖进输入区（作品 ID 保持
   默认 `demo_work`，章节 ID 自动填为 `0001`，最大返工保持 2），点击
   “开始翻译”。未配置 API key 时走 `DRAFT:` 离线回退，仍可完整观察六个
   Agent 的状态流转、LLM 流式输出、QA 结果与最终译文；若配置了真实 key，
   需额外确认结果与事件中未出现 `DRAFT:` / `llm.fallback` 回退。
4. 对照下面四组清单逐项勾选；任一项不通过，先截图记录现象再反馈。

## 一、视觉

- [ ] 页面整体为浅色系：背景为浅灰白 `#f7f8fa`，面板纯白并以细边框分隔，
      无深色背景、无霓虹光效。
- [ ] 页面加载后 header 为白色 sticky 顶栏，向下滚动时始终可见；标题、
      连接状态、运行状态、运行选择器齐全。
- [ ] 主强调色统一为蓝色 `#2563eb`：“开始翻译”主按钮为蓝色实心，其他
      强调元素（选中态、运行中标识等）使用同一蓝色。
- [ ] 六张 Agent 卡片（调度 / 术语 / 翻译 / 审校 / 润色 / QA 终审）以
      状态点徽章区分 waiting 灰 / running 蓝 / completed 绿 / failed 红。
- [ ] 面板标题、正文、辅助文字层级清晰，浅色背景下对比度充足、易读。
- [ ] 事件时间线每行时间 / 来源 / 事件类型对齐，failed 类事件标红、
      completed 类事件标绿。
- [ ] 运行概览中 Run、作品 / 章节、QA、返工四项指标排布整齐，QA 判定与
      分数突出显示。

## 二、交互

- [ ] 粘贴文本后字符计数（“N 字符”）实时更新。
- [ ] 拖入或用“选择 TXT”按钮载入 UTF-8 TXT 后：内容进入输入框、字符
      计数更新、章节 ID 自动填充为文件名（去掉 `.txt`），并出现“已载入”
      成功提示；拖入悬停时输入区有高亮反馈。
- [ ] 章节 ID 已手动填写时，再载入文件不会被文件名覆盖。
- [ ] 拖入非 TXT 文件时提示“请选择 TXT 文本文件。”，且已输入内容不被
      清空。
- [ ] “清空”按钮清空原文、重置字符计数与最终译文占位文案。
- [ ] 点击“开始翻译”后按钮立即禁用，任务完成或失败后恢复可用。
- [ ] 点击任一 Agent 卡片，LLM 流式输出切换为该 Agent 的增量内容。
- [ ] header 运行选择器可切换到历史运行，并重现其 Agent 状态、流式输出
      与事件时间线。

## 三、状态与异常

- [ ] SSE 连接成功时连接状态显示“● 实时连接”；断线时显示“重连中”，
      恢复后自动回到实时状态。
- [ ] 提交任务后运行状态由“等待运行”变为 running，完成后变为
      completed，失败时变为 failed。
- [ ] running 状态的 Agent 卡片为蓝色脉冲点，尚未轮到的卡片保持 waiting
      灰点。
- [ ] Agent 完成后状态点变绿（失败变红），卡片显示耗时（ms）与模型档位。
- [ ] 空文本点击“开始翻译”时提示“请粘贴原文或拖入一个非空 TXT 文件。”，
      不发起请求。
- [ ] 任务运行中另开窗口打开工作台，提示“检测到运行中的任务”，按钮保持
      禁用并自动接续显示进度（单任务并发限制）。
- [ ] 任务失败时任务消息显示具体错误原因，“开始翻译”按钮恢复可用。
- [ ] 翻译完成后“最终译文”显示译文全文，元信息含输出路径与 QA 判定 /
      分数；事件时间线末尾出现 run.completed 事件。

## 四、响应式

- [ ] 宽屏（≥900px）时左主列与右侧事件时间线双栏并排，主列更宽。
- [ ] 窗口收窄到 <900px 时自动变为单栏，事件时间线移到主列下方。
- [ ] 窄屏下作品 ID / 章节 ID / 最大返工三个输入字段纵向堆叠，页面无横向
      滚动条。
- [ ] 窄屏下六张 Agent 卡片自动换行 / 堆叠，不溢出面板边界。
- [ ] 流式输出与最终译文区域在任意宽度下保持固定高度与内部滚动，内容不
      被截断、不撑破布局。
- [ ] 拖动窗口边缘连续改变宽度时布局平滑切换，无内容重叠或遮挡。
