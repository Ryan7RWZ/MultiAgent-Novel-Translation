# 浏览器翻译制作台验收清单

适用范围：`src/mant/observability/dashboard.html` 单页翻译制作台。页面保持零前端
构建依赖，功能与后端接口继续由 `src/mant/observability/dashboard.py` 提供。

## 验收方法

1. 在仓库根目录执行 `cmd /c start_mant.bat --check`，确认配置、导入和 Python
   环境自检通过（不会打印密钥）。
2. 双击 `start_mant.bat`，浏览器应自动打开 `http://127.0.0.1:8765`。
3. 把一个已授权的 UTF-8 TXT 拖进输入区，填写作品/章节 ID，点击“开始翻译”。
   真实供应商验收前仍须单独确认数据发送授权与费用；普通界面验收应使用 fake 或
   离线回退，避免意外付费。
4. 对照下列清单逐项检查；失败项应记录窗口宽度、运行 ID、截图和复现步骤。

## 一、视觉与信息架构

- [ ] 顶部 sticky 状态栏在滚动时始终可见，品牌、实时连接、任务状态和历史运行
      选择器齐全，无横向溢出。
- [ ] 页面首屏清楚区分“原文与任务”和“运行概览 / 协作链路”，首次使用者无需
      滚动即可找到提交入口与状态反馈。
- [ ] 总进度条显示当前阶段、百分比和片段进度；等待、运行、完成、失败四种状态
      的文案和颜色可区分。
- [ ] 运行概览正确展示片段数、模型调用数、Token 总量、耗时、运行 ID、作品/
      章节以及 QA 判定。
- [ ] 六张 Agent 卡片按编排、术语、翻译与返工、审校、润色、终审排列，并用灰/
      蓝/绿/红状态点区分等待、运行、完成、失败。
- [ ] 深色模型输出区与浅色事件/结果区层级明确；正文、辅助文案和等宽技术信息
      均清晰可读。
- [ ] 最终译文采用适合长文阅读的行高，长文本在面板内滚动且不撑破页面。

## 二、输入与任务交互

- [ ] 粘贴文本后字符计数实时更新并带千位分隔。
- [ ] 拖入或点击“载入 TXT”选择 UTF-8 `.txt` 后，文本进入输入框；章节 ID 为空
      时自动使用文件名，已有章节 ID 不被覆盖。
- [ ] 拖入非 TXT 文件时显示明确错误，原有文本保持不变；拖拽悬停时输入框高亮。
- [ ] 空文本点击“开始翻译”只显示校验提示，不发送请求。
- [ ] 任务提交后翻译、清空和文件按钮立即禁用；完成或失败后恢复。
- [ ] “清空”重置原文、字符计数、文件选择和结果占位，不修改历史运行数据。
- [ ] 任务运行中打开第二个页面时，页面识别活动任务、自动轮询并保持提交入口禁用。

## 三、运行观测

- [ ] SSE 连接成功显示“实时连接”，断开显示“正在重连”，恢复后自动回到连接态。
- [ ] Agent 状态、完成数和异常数随事件更新；运行中的蓝点有脉冲动画（系统启用
      reduced motion 时允许无动画）。
- [ ] 点击 Agent 卡片或输出区角色标签，只显示该角色的调用；调用选择器可继续按
      片段与返工轮次切换，多个并发调用的 token 不串接。
- [ ] 大量 `llm.token` 不出现在事件记录中，但对应模型文本持续增长，页面滚动与
      点击保持响应。
- [ ] 事件记录按时间倒序显示，完成标绿、异常标红、路由单独标识；“异常”筛选只
      保留失败、重试、降级、完整性和预算/熔断事件。
- [ ] `output.integrity_failed` 显示片段编号与完整度比例，便于解释“调用成功但
      工作流未放行”的情况。
- [ ] header 历史运行选择器能重放已接收运行的 Agent 状态、模型输出和事件记录。

## 四、结果交付

- [ ] 任务完成后显示完整译文，元信息含输出路径、QA 判定、分数和覆盖率。
- [ ] “复制译文”写入剪贴板并显示成功提示；浏览器拒绝权限时显示可操作的失败提示。
- [ ] “下载 TXT”生成以章节 ID 命名的 UTF-8 文本；无结果时两个按钮均不可用。
- [ ] QA `rework` 时顶部显示“流程完成 · 待返工”，不能误写成质量通过。
- [ ] 后台失败时展示具体错误，原文仍保留，用户可以修改后重新提交。

## 五、响应式与可访问性

- [ ] ≥1120px 时任务区、运行区双栏，模型输出与事件记录双栏；主要内容宽于辅助栏。
- [ ] 760–1119px 时所有主要面板折叠为单栏，事件记录位于模型输出之后。
- [ ] <760px 时表单、指标和 Agent 卡片合理换行，按钮可点击，页面无横向滚动条。
- [ ] Tab 可依次访问输入、按钮、选择器、Agent 卡片和筛选器；焦点轮廓清晰。
- [ ] Enter/Space 能激活 Agent 卡片；状态提示使用 `aria-live`，图标不重复朗读。
- [ ] 连续缩放窗口、切换长运行 ID 和加载长文本时，无内容重叠或布局跳变。
